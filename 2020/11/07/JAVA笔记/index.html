运算符

自增自减

一般都是用在某个整数类型变量 增加 1 或 减少 1，Java 特殊运算符，用于这种表达式，叫做自增运算符 ++ 和自减运算符 – 。

int a=0,i=0;
int a = i++;        //a = 0, i = 1
a = ++i;        //a = 1, i = 1
a = i--;        //a = 0, i = -1
a = --i;        //a = -1, i = -1

Switch…Case语句

int a=0;
switch(a){
case 0 :
    a++;  //语句1
case 1 :
    ++a;  //语句2
case 2 :
    a++;  //语句3
default :
    break;  //语句
}

default可有可无
作用是在case都检索完
没有找到对应的条件
才执行default下的语句

Java 面向对象

抽象

关键字 : abstract

抽象起到规范的作用 实现多个子类中共用的部分 不用重复写到实现类中

如类中有抽象化的方法 类必须抽象化

被抽象化的方法 不能有方法体

public abstract class 类名{
    修饰符 abstract void 方法名();
}

接口

接口中所有的方法都是抽象方法

接口中所有的内容都是公共的 所有的变量都是全局静态变量

接口使用 interface 来声明 不再是类

与类的区别是 类只能单继承 接口支持多实现

接口同样具有多态性

接口可以把多种不相关的内容进行整合

public interface 类名{
    void 类名();                  //方法一
    public abstract void 方法名();  //方法二 此方法可以省略
}

工具类

Arrays工具类

toString方法

实现数组的遍历,并以一个美观的形式输出在控制台

int[] arr = new int[]{1,2,3,4,5};
System.out.println(Arrays.toString(arr));

sort方法

实现把数组升序排序

int[] arr = new int[]{5,4,3,2,1};
Arrays.sort(arr)
System.out.println(Arrays.toString(arr));

二分查找法

输出在数组中查找需要查找的数的索引值
必须在有序的数组下查找 找不到则返回负数

int[] arr = new int[]{5,4,3,2,1};
Arrays.sort(arr)
System.out.println(Arrays.binarySearch(arr,3));

复制

int[] arr = new int[]{5,4,3,2,1};
int[] newArr=Arrays.copyOf(arr,6); //复制数组，不足的部分0填充
System.out.println(Arrays.toString(arr));

int[] newArr02=Arrays.copyOf(arr,2,4); //复制[2,4) 区间形式
System.out.println(Arrays.toString(arr));

比较

int[] arr1 = new int[]{5,4,3,2,1};
int[] arr2 = new int[]{5,4,3,2,1};

System.out.println(arr1==arr2); //false 比较的是左右两侧的地址的值
System.out.println(Arrays.equals(arr1,arr2)); //true 比较的数组元素具体的值

填充

int[] arr = new int[]{5,4,3,2,1};

Arrays.fill(arr,0);
System.out.println(Arrays.toString(arr)); //全部填充为0

Arrays.fillI(arr,2,1);
System.out.println(Arrays.toString(arr)); //[0,2)填充为1

int[] arr = new int[]{5,4,3,2,1};
int[] arr1 = new int[]{0,0,0,0,0};
System.out.arraycopy(arr,2,arr1,3,4);
